Kakai Runnable Server & Web Framework

Kakai is a standalone runnable server built with everything you need to create a full stack java application. Kakai handles dependency injection, http routing, data persistence and data binding.

Quick Start

Kakai can be bootstrapped in a matter of seconds. A quick walk through using gradle.

Gradle

            
implements "io.kakai:kakai:1.0"

A main entry point for our application.

Main.java

            
import io.kakai.Kakai;

public class Main {
    public static void main(String[] args){
        new Kakai(8080).start();
    }
}

A Router is synonymous with Controller in the MVC dialect. A router can handle many endpoints. Example:

HelloRouter.java

            
import io.kakai.annotate.Router;

@Router
public class HelloRouter {

     @Text
     @Get("/")
     public String hello(){
        return "hello world.";
     }

}
