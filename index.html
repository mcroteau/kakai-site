>_ Kakai runnable

Kakai Runnable Server & Web Framework

Beta Release v1.0

Kakai standalone runnable server contains everything you need to create a full stack java application. Kakai handles dependency injection, http routing, data persistence and page data binding much like the jsp taglib.

Quick Start

Kakai can be bootstrapped in a matter of seconds. A quick walk through using gradle.

Gradle

            
implementation "io.kakai:kakai:1.0"

A main entry point for our application.

Main.java

            
import io.kakai.Kakai;

public class Main {
    public static void main(String[] args){
        new Kakai(8080).start();
    }
}

A Router is synonymous with Controller in the MVC world. A router can handle many endpoints. Example:

HelloRouter.java

            
import io.kakai.annotate.Router;

@Router
public class HelloRouter {

     @Text
     @Get("/")
     public String hello(){
        return "hello world.";
     }

}

Then its as simple as running the following command.

Run it!

            
gradle run

Browse:

            
http://localhost:8080/
